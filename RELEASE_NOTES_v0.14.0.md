# FlyPPTTimer v0.14.0

## 远程演示管理

- 手机遥控在同一链接中增加结束放映、关闭当前受控文稿、退出 PowerPoint、强制退出全部 PowerPoint/WPS/演示软件四种明确操作。
- FlyPPTTimer 打开的 PowerPoint 文稿使用只读请求并记录受控路径；不会静默关闭用户自行打开的文稿。
- 演示命令通过 PowerPoint STA 队列异步运行，状态包含操作编号、阶段和提示，避免 HTTP 请求等待 COM。
- WPS 检测基于真实进程能力显示；未验证的 COM 文稿操作不会伪装可用。

## 设置与界面

- 设置编辑改为草稿应用：校验成功才原子应用到运行配置和手机状态；取消不影响已生效的文件规则。
- 文件规则区统一使用浅色卡片、柔和选择色和危险操作色；重复路径和无效时长会被拒绝。
- 修复地址选择色块菜单被提前释放的问题，菜单可正常点击、键盘打开并受屏幕工作区约束。
- 设置页签在窄窗口会自动换行，设置窗口和远程控制窗口均允许调整尺寸。

## 已知人工验证

- Microsoft PowerPoint 及 WPS 的真实 COM/进程行为需要在安装了对应软件的本机完成集成测试。
- 首次打开 PowerPoint 预计需要 1–3 秒，手机端会显示进度而不是阻塞页面。
