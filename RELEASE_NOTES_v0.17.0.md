# FlyPPTTimer v0.17.0

v0.17.0 让台下控制计时器成为完整工作流。手机计时页现在可以直接编辑时、分、秒并切换倒计时或正计时；修改会立即保存到电脑端配置，已打开的 PC 设置窗口也会同步刷新。

PC“时长设置”新增倒计时结束策略：可以在到达零点后停止，也可以继续计时。继续模式会使用已配置的超时颜色显示超出的时长，并保留暂停、恢复等计时操作；手机端同时显示醒目的超时状态。

“其他设置”现在包含完整的项目与作者介绍、联系邮箱和 GitHub 项目入口，欢迎用户参与测试、反馈和共同开发。

本版本继续保留 v0.16.3 的设置规则状态文案完整显示修复，以及 v0.16.2 的 Per-Monitor V2 DPI、响应式远控布局和窗口位置记忆。
