# FlyPPTTimer v0.20.0

v0.20.0 汇总了 v0.18.9 正式版之后的演示控制、手机遥控、界面稳定性和版本更新能力，是当前推荐的正式版本。

## 新版本检测与安装

- “其他设置”新增“启动时检测新版本”，默认关闭，不会在用户未选择时主动联网检测。
- 系统托盘右键菜单新增“检测新版本”，支持随时从 Gitee Release 手动检查。
- 发现新版本后显示版本号和更新说明，由用户决定是否继续。
- 安装版确认后下载 Windows x64 安装程序；存在 SHA-256 附件时会先验证文件完整性，再在当前程序退出后启动覆盖安装。
- 覆盖安装继续保留原有配置文件；新增选项沿用默认值，用户可在设置中自行选择。
- 绿色便携版不会自动覆盖文件，确认后直接打开对应的 Gitee Release 页面供用户选择下载。

## 手机遥控

- “计时 / 演示”支持整页任意角度左右滑动；滑动过程跟手、松手平滑吸附，并可由新的滑动或标签点击立即打断和接管。
- 手机可修改时长与正计时/倒计时；修改全局时长时可选择是否同步到全部文稿规则。
- 计时与演示页面均可退出“时间到”黑屏，并实时显示电脑静音状态。

## PowerPoint / WPS 与文稿规则

- 外部打开的文稿也可被识别、放映和按最后打开顺序逐个关闭；FlyPPTTimer 不再因临时放映范围设置导致文稿出现保存提示。
- 精简重复的演示软件退出入口，并改善 WPS 首次打开时的窗口最大化行为。
- 文件规则支持多选和批量修改时长、正计时/倒计时方式。

## 桌面界面与配置

- 默认计时窗口调整为 100×35，继续保持时间文字水平、垂直居中，并以配置的默认点位为固定中心缩放。
- 修复 PC 远程控制页面布局异常；远程窗口继续支持多 DPI、响应式布局和窗口状态记忆。
- “其他设置”补充 GitHub、Gitee、作者信息和项目说明。
- 安装升级保留原配置；首次出现的新选项采用安全默认值，不替用户自动启用。

## 下载文件

GitHub Release 提供以下四个附件，Gitee 镜像按现有同步配置获取：

- `FlyPPTTimer-v0.20.0-setup-win-x64.exe`
- `FlyPPTTimer-v0.20.0-setup-win-x64.exe.sha256`
- `FlyPPTTimer-v0.20.0-portable-win-x64.zip`
- `FlyPPTTimer-v0.20.0-portable-win-x64.zip.sha256`

安装版适合普通用户直接覆盖升级；便携版解压即用。建议同时下载对应 `.sha256` 文件校验完整性。
