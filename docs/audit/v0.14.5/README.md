# v0.14.5 审计记录

## 问题依据

用户实测表明远程控制窗口长期存在字号、控件高度和滚动层级不一致：标题、状态、按钮、地址选择和规则状态使用不同视觉尺度；演示文稿页的内外滚动导致下方操作难以到达。

## 修复边界

- 自定义远控窗口及其菜单、确认窗口统一为微软雅黑 10pt。
- 交互色块统一引用 `ModernTheme.StandardControlHeight`，不再分散使用 36、40 等固定高度。
- 演示页仅外层 `AutoScroll`，规则列表关闭内部滚动并将滚轮路由至页面。
- 不修改计时、PowerPoint COM、手机协议、文件规则、token、端口、托盘或多屏逻辑。

## 人工验收边界

必须使用 `FlyPPTTimer-v0.14.5-windows-x64` CI Artifact 验证 100%、125%、150% DPI、默认/最小/放大窗口、空列表与多规则列表，以及真实 PowerPoint/WPS、手机、双屏和演讲者视图。
