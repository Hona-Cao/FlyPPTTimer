# v0.14.3 审计记录

## 问题依据

用户在真实 DPI 环境的截图显示，远控窗口虽然可被拉大，但标题、副标题、服务状态、端口输入和规则行仍被固定行高裁切。该问题不应通过继续扩大窗口解决。

## 修复边界

- 顶栏、状态栏、导航与底部操作改用 `AutoSize` 行。
- 按钮、输入宿主和规则卡按字体测量最小高度。
- 规则列表为三条规则保留最小内容高度，空间不足由滚动容器承担。
- 未触碰计时、PowerPoint COM、手机远控、文件规则匹配、端口、token 或任务栏逻辑。

## 人工验收边界

必须以 `FlyPPTTimer-v0.14.3-windows-x64` CI Artifact 检查 100%、125%、150% DPI。未获得真实截图前，不提交或伪造验收图片。
