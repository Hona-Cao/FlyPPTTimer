# v0.14.4 审计记录

## 问题依据

真实 DPI 截图显示 v0.14.3 的远程控制窗口仍存在固定区域过高、连接页链接区域裁切、演示页外层滚动和跨页底部操作区占用空间的问题。

## 修复边界

- 顶部仅保留紧凑标题区、单行服务状态区和导航区。
- “复制放行命令”仅保留在远程连接页。
- 远程连接页只有一个滚动容器，面板高度遵循内容而不是填满视口。
- 演示文稿页移除外层滚动，规则列表占主要弹性空间；规则编辑和放映操作使用可展开卡片。
- 不修改计时、PowerPoint COM、手机远控、文件规则、端口、token 或托盘逻辑。

## 人工验收边界

必须使用 `FlyPPTTimer-v0.14.4-windows-x64` CI Artifact 检查 100%、125%、150% DPI、默认/最小窗口、PPT/WPS、手机、双屏与演讲者视图。未获得真实截图前，不提交或伪造验收图。
