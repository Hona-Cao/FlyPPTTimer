可借鉴逻辑：
- 使用轮询检测前台窗口是否覆盖当前屏幕工作区域，作为全屏放映的基础判断。
- 使用进程名/窗口排除或白名单限制自动计时触发范围。
- 悬浮窗口置顶、无边框、鼠标穿透适合演讲场景。
- 配置文件放在程序目录，便于便携迁移。
- 支持手动热键、托盘菜单、多显示器位置记忆。
- 提前提醒和结束提醒应与计时状态解耦，避免 UI 计时器漂移影响实际计时。

本项目不直接复用的内容：
- 不复用 AHK 源码、图标、图片和声音资源。
- 不使用 AHK 编译链作为主实现。
- 不复制旧项目 UI，仅参考自动检测和配置思路。
