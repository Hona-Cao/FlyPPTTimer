# v0.16.2 办公室多屏 DPI 验收清单

当前开发机不具备任务指定的全部 125%/150% 双屏组合时，以下项目必须在办公室设备上逐项确认，未执行项不得标记通过。

## 环境

- 电脑 A：2560×1440，125%。
- 电脑 B：2560×1600，150%。
- 电脑 B 扩展屏：2560×1440，125%。

## 操作

1. 在 125% 显示器打开远控窗口，分别检查远程连接页和演示文稿页。
2. 调整窗口位置和尺寸，关闭并重启应用，核对显示器、位置和逻辑客户区尺寸。
3. 将窗口从 125% 拖到 150%，再从 150% 拖回 125%。
4. 在两个屏幕分别执行最大化、还原并重启应用。
5. 分别检查最小窗口、默认窗口和最大化窗口。

## 每一步检查

- 标题和正文比例一致，正文没有随窗口宽度连续缩放。
- 侧栏、卡片、行高、端口、按钮、地址选择器和二维码比例正确。
- 卡片不重叠、不被压扁；“当前页放映”和危险操作完整。
- Standard 模式右侧无滚动；Compact 模式只在规定区域出现滚动。
- 连接页紧凑按钮为两行，演示文稿放映按钮为 2×2。
- 截图分别保存到 `artifacts/v0.16.2/dpi/125`、`150` 和 `cross-monitor`。
