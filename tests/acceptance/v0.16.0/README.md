# v0.16.0 人工验收

## 远程连接

- 导航只有一行文字。
- 所有按钮只有一行。
- 端口、重启和关闭不重叠。
- 地址和链接完整可操作。
- 二维码不变形。
- 默认窗口无整体滚动。
- 高 DPI 下可滚动到全部内容。

## 演示文稿

- 右侧没有滚动条。
- 规则编辑、放映、危险操作同时可见。
- 只有左侧列表滚动。
- 空列表显示“暂无演示文稿”。
- 长文件名和路径使用省略号，不挤压按钮。
- “更多”“保存”“启用/禁用”保持单行。
- 四个放映按钮保持单行。
- 两个危险按钮保持单行。

## 功能

- 添加、删除、刷新正常。
- 启用/禁用、保存时长正常。
- 复制路径、显示文件正常。
- 打开、从头放映、当前页放映、结束放映正常。
- 退出与强制退出确认正常。
- 手机端协议和功能无变化。
