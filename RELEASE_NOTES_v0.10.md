# FlyPPTTimer v0.10

Windows 便携版 PPT/演讲计时器。

## 下载

- `FlyPPTTimer_Setup_v0.10.exe`：安装版，安装到当前用户目录，不需要管理员权限。
- `FlyPPTTimer_Portable_v0.10.zip`：绿色版，解压后运行 `FlyPPTTimer.exe`。

## 主要功能

- PowerPoint/WPS 放映自动开始计时。
- 多屏计时器窗口同步显示。
- 默认显示分秒，超过 1 小时自动显示小时。
- 手机网页遥控，支持开始、暂停、继续、停止并重置、显示状态。
- 设置窗口支持外观、位置、快捷键、远程控制和 PPT 文件规则。
- 可为不同 PPT 文件设置不同计时时长。

## 注意

- 首次启动会自动生成本机远程控制 token。
- 手机遥控需要手机和电脑在同一网络下。
- 如果手机无法访问，请检查 Windows 防火墙和当前网络类型。
