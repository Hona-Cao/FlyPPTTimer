# FlyPPTTimer v0.16.0

## 范围

v0.16.0 完全重写 PC 远程控制窗口的 UI 外壳。计时、PowerPoint/WPS、手机协议、远程服务、token、端口和规则模型保持不变。

## 界面

- 采用纯文字侧栏，导航只保留“远程连接”和“演示文稿”。
- 删除按钮、导航和状态中的功能图标与装饰符号。
- 所有按钮强制单行显示。
- 远程连接页压缩状态与说明，保留二维码、地址、链接和三项访问操作。
- 演示文稿页固定为左侧列表、右侧详情。
- 右侧规则、放映和危险操作默认同时可见。
- 只有演示文稿列表滚动。
- 菜单和反馈文字统一缩短。

## 未修改

- 计时状态机
- PowerPoint/WPS 控制服务
- 手机网页与协议
- token 和端口
- 配置与规则模型
- 托盘、多屏和快捷键
- Logo、应用图标和托盘图标

## 验收

自动化构建和测试结果应在应用材料包后真实填写。100%、125%、150%、200% DPI，真实 Office/WPS、手机、双屏和演讲者视图仍需人工验收。
