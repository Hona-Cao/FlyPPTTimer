# FlyPPTTimer v0.19.0

本版集中改善紧凑显示、手机操作和外部演示文稿控制。

## 主要更新

- 修复 PC 远程控制窗口侧栏占满整行、主内容被挤到下方的布局回归。
- 新安装默认使用 100×35 计时窗口，时间继续围绕设置中的固定中心原点显示。
- 手机“计时 / 演示”支持左右滑动与点击两种切换方式。
- 外部打开的 PowerPoint 文稿也能从 FlyPPTTimer 逐个关闭：最后打开的先关闭，且不显示保存询问。
- 删除重复的普通退出入口，统一保留“退出演示软件”。
- 修改全局计时时长时，PC 和手机都会询问是否同步到全部待控文件规则。
- 软件内加入 GitHub、Gitee 两个项目主页入口，并更新作者信息。

## 下载文件

- `FlyPPTTimer-v0.19.0-setup-win-x64.exe`：Windows x64 安装版。
- `FlyPPTTimer-v0.19.0-portable-win-x64.zip`：Windows x64 便携版。

建议在正式活动前，用实际 PowerPoint/WPS 版本完成一次打开、放映、关闭和手机遥控彩排。强制“退出演示软件”会终止全部相关进程，使用前请确认其他文稿没有未保存修改。
