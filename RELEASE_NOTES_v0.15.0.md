# FlyPPTTimer v0.15.0

## PC 远程控制窗口重设计

- 使用独立的远控仪表盘主题重建 PC 远程控制窗口，不修改其他桌面窗口的共享主题。
- 左侧固定导航在“远程连接”和“演示文稿”之间切换，主内容区不再依赖顶部页签或纵向长页面。
- 远程连接页重新组织服务状态、端口、二维码、访问地址、浏览器打开和防火墙命令。
- 演示文稿页采用左侧规则列表、右侧详情与操作的主从式双栏布局，使规则和操作同时可见。

## 保持不变

- 计时状态机、PowerPoint/WPS 控制服务、手机端协议与网页、token、端口、配置/规则模型、托盘、多屏、热键、计时悬浮窗、Logo 和图标均未修改。
- 继续使用 .NET 8 与 Windows Forms，不引入第三方 UI 框架。
