# FlyPPTTimer v0.13.0 发布候选说明

v0.13 是桌面端的 WinUI 3 完整重构版本。旧 v0.12.1 WinForms 项目和发布目录均保留。

主要变化：Fluent NavigationView 主界面、独立多屏计时悬浮窗、统一 Core 状态源、PowerPoint STA 基础设施、完整文件规则交互、异步首张幻灯片预览、同步增强的手机演示列表，以及新的自包含绿色版与安装版脚本。

本版本仍只正式支持 Microsoft PowerPoint 遥控。提交到主分支或公开发布前，请完成 `TEST_REPORT.md` 中列出的 Office、DPI、Clash/TUN 和 8 小时人工验收。
